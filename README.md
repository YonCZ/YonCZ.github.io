I used a Large Language Model (Gemini) as a supportive tool for this project in the following areas:

Conceptual Brainstorming: I used the model to brainstorm various portfolio styles and layouts that would align with my desired monochromatic theme.

Workflow Planning: The AI helped me establish a structured development plan, suggesting which components and technical layers to prioritize first.

Code Optimization: Towards the end of the project, I used the model to identify redundancies and "ghost selectors" in my CSS, and to help polish and standardize the code comments.

Guideline Compliance: I used the model as a secondary check to ensure the final implementation met all the technical guidelines specified in the project assignment.
